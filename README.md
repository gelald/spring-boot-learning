## 模块说明

### lifecycle

演示 Spring Bean 的初始化方法和销毁方法的各种配置方式及其执行顺序

### filter

演示 SpringBoot 中如何定义与注册 Filter 过滤器

### interceptor

演示 SpringBoot 中如何定义与注册 HandlerInterceptor 拦截器

### datetime

演示 SpringBoot 中针对 Date 和 LocalDateTime 系日期类型数据的序列化与反序列化的方式
